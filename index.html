Estudando Git e GitHub

Informações

  • Git é um sistema de Controle de versão
  • Lidar com git e comandos
  • Como subir código no GitHub
  • Git tira fotos (snapshots) dos estados dos arquivos
  • GitHub ferramenta na nuvem que armazena projetos versionados em Git

Comandos

  • git config --global user.name "Tiago"
  • git config --global user.email "tiago.bezesan@gmail.com"
  • git config --global core.editor brackets
  • git config user.name
  • git config --list
  • git init

Ciclo de Vida

Estados

  • untracked: momento em que o arquivo foi adicionado ao repositório, mas ainda não é visto pelo git
  • unmodified: existe no git, mas não teve nenhuma modificação.
  • modified: arquivo modificado
  • staged: local em o arquivo é colocado para ser versionado

Mais comandos

  • git status
  • git add NomeArquivo: adiciona para staged, ficando pronto para ser comitado
  • git commit -m "Um comentário": momento em que avisamos ao Git para pegar todos os arquivos do repositório e criar uma imagem (snapshot)
  • git log
  • git shortlog
  • git show HASH
  • git diff (sempre usar antes de um commit)
